│       │  GNOME 48, quoi de neuf ?  │       │
│ ┌─────┴────────────────────────────┴─────┐ │
│ │                                        │ │
│ │ Le 19 mars, le projet GNOME présentait │ │
│ │ son nouveau bébé : GNOME 48, une       │ │
│ │ nouvelle version bourrée de            │ │
│ │ nouveautés. De l'amélioration du       │ │
│ │ centre de notifications à              │ │
│ │ l'introduction de l'HDR, voici un      │ │
│ │ résumé des principales nouveautés :    │ │
│ │                                        │ │
│ │ Empilement des notifications :         │ │
│ │ Cette mise à jour commence par le      │ │
│ │ centre de notifications avec une       │ │
│ │ nouvelle fonctionnalité vous           │ │
│ │ permettant de grouper les              │ │
│ │ notifications par application.         │ │
│ │ Cela améliore l'organisation           │ │
│ │ et facilite la navigation.             │ │
│ │                                        │ │
│ │ Mise à jour de la visionneuse          │ │
│ │ d'images :                             │ │
│ │ GNOME met à jour sa visionneuse        │ │
│ │ d'images par défaut, introduite dans   │ │
│ │ la version 45. Elle ajoute une         │ │
│ │ fonction d'édition d'images simple,    │ │
│ │ permettant de rogner, recadrer et      │ │
│ │ faire pivoter une image. Cette version │ │
│ │ améliore également la prise en charge  │ │
│ │ des métadonnées et du format RAW.      │ │
│ │                                        │ │
│ │ Bien-être numérique:                   │ │
│ │ GNOME introduit une nouvelle           │ │
│ │ fonctionnalité accessible dans les     │ │
│ │ paramètres, permettant de visualiser   │ │
│ │ le temps passé sur son appareil.       │ │
│ │ Un historique des jours et semaines    │ │
│ │ précédents est disponible, ainsi       │ │
│ │ qu'une option pour limiter son temps   │ │
│ │ d'utilisation avec une durée           │ │
│ │ journalière définie.                   │ │
│ │                                        │ │
│ │ HDR:                                   │ │
│ │ GNOME introduit enfin le HDR           │ │
│ │ (High Dynamic Range) au niveau du      │ │
│ │ système, permettant aux utilisateurs   │ │
│ │ possédant un écran compatible          │ │
│ │ d'afficher du contenu en HDR dans les  │ │
│ │ applications qui le prennent en        │ │
│ │ charge. Cependant, le nombre           │ │
│ │ d'applications compatibles reste       │ │
│ │ limité.                                │ │
│ │                                        │ │
│ │ Autres nouveautés:                     │ │
│ │ Au-delà des améliorations de           │ │
│ │ performances et des corrections de     │ │
│ │ bugs, GNOME change sa police           │ │
│ │ d'interface et de terminal par         │ │
│ │ défaut pour Adwaita Sans et            │ │
│ │ Adwaita Mono.                          │ │
│ │ Plusieurs nouvelles applications       │ │
│ │ accompagnent cette version :           │ │
│ │                                        │ │
│ │ · Audio Player :                       │ │
│ │   - Une application minimaliste        │ │
│ │     de lecture de fichiers audio.      │ │
│ │                                        │ │
│ │ · Drum Machine :                       │ │
│ │   - Un programme simple permettant la  │ │
│ │     création et l'écoute de            │ │
│ │     "Drum Patterns".                   │ │
│ │                                        │ │
│ │ · Iotas :                              │ │
│ │   - Une application de prise de notes  │ │
│ │     rappelant Notion.                  │ │
│ │                                        │ │
│ │ · Keypunch :                           │ │
│ │   - Un programme d'entraînement à la   │ │
│ │     dactylographie.                    │ │
│ │                                        │ │
│ │ · Exercise Timer :                     │ │
│ │   - Une application minimaliste pour   │ │
│ │     créer des minuteurs pour le sport. │ │
│ │                                        │ │
